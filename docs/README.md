# TTPE Documentation - Additional Assets

This directory contains static assets for the documentation site.

## Structure

- `img/` - Images, logos, screenshots
- `fonts/` - Custom fonts (if needed)
- `downloads/` - Downloadable resources

## Adding Images

Place images in `img/` and reference them in markdown:

```markdown
![Alt text](/img/your-image.png)
```

## Logo

The TTPE logo is at `/img/logo.svg`. Update it to match your branding.

## Favicon

Place `favicon.ico` in this directory. Docusaurus will use it automatically.
